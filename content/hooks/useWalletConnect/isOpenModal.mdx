# isOpenModal

Boolean indicating whether the connection modal is open.

## Property

`isOpenModal: boolean`

## Type

`boolean` - `true` if modal is open, `false` otherwise

## Usage

```tsx
import { useWalletConnect } from "@dogeos/dogeos-sdk";

function ModalControls() {
  const { openModal, closeModal, isOpenModal } = useWalletConnect();

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      {isOpenModal && (
        <div>
          <p>Modal is open</p>
          <button onClick={closeModal}>Close Modal</button>
        </div>
      )}
    </div>
  );
}
```

## Related Methods

- `openModal()` - Opens the modal
- `closeModal()` - Closes the modal

## Related Properties

- `isConnected` - Connection status
- `isConnecting` - Connection in progress status
