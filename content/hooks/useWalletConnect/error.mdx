# error

Error message if an error occurred during connection or other wallet operations.

## Property

`error: string`

## Type

`string` - Error message if an error occurred, empty string otherwise

## Usage

```tsx
import { useWalletConnect } from "@dogeos/dogeos-sdk";

function ConnectionStatus() {
  const { error, openModal, isConnected } = useWalletConnect();

  if (error) {
    return (
      <div>
        <p>Error: {error}</p>
        <button onClick={openModal}>Try Again</button>
      </div>
    );
  }

  return (
    <div>
      Status: {isConnected ? "Connected" : "Not Connected"}
    </div>
  );
}
```

## Notes

- Returns an empty string if no error has occurred
- Error is cleared when a new connection attempt is made
- Common errors include:
  - User rejection
  - Network errors
  - Wallet not available
  - Chain not supported

## Related Properties

- `isConnected` - Connection status
- `isConnecting` - Connection in progress status
