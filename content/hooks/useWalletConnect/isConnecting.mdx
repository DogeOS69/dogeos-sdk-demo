# isConnecting

Boolean indicating whether a connection is in progress.

## Property

`isConnecting: boolean`

## Type

`boolean` - `true` if connection is in progress, `false` otherwise

## Usage

```tsx
import { useWalletConnect } from "@dogeos/dogeos-sdk";

function ConnectionButton() {
  const { openModal, isConnecting, isConnected } = useWalletConnect();

  if (isConnecting) {
    return <button disabled>Connecting...</button>;
  }

  return (
    <button onClick={openModal}>
      {isConnected ? "Connected" : "Connect Wallet"}
    </button>
  );
}
```

## Related Properties

- `isConnected` - Connection status
- `isOpenModal` - Modal open status
- `error` - Error state if connection fails
